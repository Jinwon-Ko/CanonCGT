# Lightroom Preset Links

This file summarizes the style IDs used to construct the supervised paired dataset in **CanonCGT**.

For supervised preset learning, we use the Expert C version of the **MIT-Adobe FiveK** dataset as the canonical target and generate preset-transformed images using Lightroom presets.

Due to license restrictions, we do **not** redistribute the Lightroom preset files or the generated paired dataset. This repository only provides source links for reproducibility. Please download each preset directly from the original provider and follow its respective license terms.
